# MERN Book Store

A simple app for viewing and adding to the book store inventory.
<br/>

# Installation

1. Clone the repo.
2. Install MongoDB
3. Install NPM Packages:
   <br/>
   `npm install`
4. NPM start in both frontend and backend directories:
   <br/>
   `NPM start`
5. Visit `localhost` to view deployment.
